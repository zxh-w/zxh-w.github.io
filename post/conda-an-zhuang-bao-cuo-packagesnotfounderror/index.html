Conda安装报错：PackagesNotFoundError

2023-04-25 1 min read # conda

PackagesNotFoundError: The following packages are not available from current channels:

otxv2==1.5.12

  • 当出现PackagesNotFoundError时，它不一定就是你的conda源出问题了，你可以先在Anaconda的官网搜一下到底conda中有没有这个包，otxv2没有在 conda提供的包里。

  • 这时候就要使用 pip工具来下载这个包了，pip install otvx2==1.5.2，虽然找到了包，但是又有了报错：

  • 这个时候在网上查了一下，输入 pip install OTXv2==1.5.12 -i http://pypi.douban.com/simple --trusted-host pypi.douban.com，安装成功


    参考来源：http://www.cnblogs.com/zengming/
