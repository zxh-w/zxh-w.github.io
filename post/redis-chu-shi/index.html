Redis初识

2023-05-06 3 min read # Redis

Redis是一个开源的，基于内存的数据结构存储，可用作于数据库、缓存、消息中间件。
Redis的全称是：Remote Dictionary Server，本质上是一个 Key-Value 类型的内存数据库。
整个数据库加载在内存当中进行操作，定期通过异步操作把数据库数据保存在硬盘。因为是纯内存操作，Redis 的性能非常出色。

Redis特性

速度快

  • 数据存在内存
  • 由C语言编写(50000 line)
  • 使用单线程模型

持久化

Redis将所有数据保持在内存中 ，对数据的更新将异步地保存到磁盘上。

多种数据结构

除了以上的数据结构，还提供了：

  • BitMaps：位图
  • HyperLogLog：超小内存唯一值计数
  • GEO：地理信息定位

支持多种语言

Java、php、Python、nodejs、lua、Rudy

简单

  • 不依赖外部库
  • 单线程模型

主从复制

高可用、分布式

  • 高可用 Redis-Sentinel(v2.8)支持
  • 分布式 Redis-Cluster(v3.0)支持

典型应用场景

  • 缓存系统
  • 计数器
  • 排行榜
  • 消息队列系统
  • 社交网络
  • 实时系统

Redis安装

Linux编译安装

  • wget http://download.redis.io/releases/redis-3.0.7.tar.gz
  • tar -xzvf redis-3.0.7.tar.gz
  • ln -s redis-3.0.7.tar.gz reids
  • make && make install

可执行文件说明

执行文件 说明
redis-server Redis服务器
redis-cli Redis命令行客户端
redis-benchmark Redis性能测试工具
redis-check-aof AOF文件修复工具
redis-check-dump RDB文件检查工具
redis-sentinel Sentinel服务器

三种启动方式

  • 生产环境最好选择配置启动
  • 单机多实例配置文件可以用端口区分开
启动方式 命令 验证
最简启动 redis-server `ps -ef
配置文件启动 redis-serve --port 6380 `netstat -antpl
动态参数启动 redis-server configPath redis-cli -h ip -p port ping

Redis常用配置
daemonize 是否是守护进程
port Redis对外端口号（默认端口6379）
logfile Redis系统日志名
dir Redis工作目录
